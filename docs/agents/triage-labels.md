# Triage Labels 规范

规范 triage 角色使用以下 label 字符串：

| 角色               | Label             |
| ------------------ | ----------------- |
| 需要维护者评估     | `needs-triage`    |
| 等待报告者补充信息 | `needs-info`      |
| 可交给 AFK agent   | `ready-for-agent` |
| 可由人类实现       | `ready-for-human` |
| 不会处理           | `wontfix`         |

除非有意调整仓库 label 词表，否则不要创建含义重复但措辞不同的 label。
