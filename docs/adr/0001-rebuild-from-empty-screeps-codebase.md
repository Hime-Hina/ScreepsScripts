# ADR 0001：从空 Screeps 代码库重建

## 状态

已接受。

## 背景

旧项目曾在单房间中存活数月，但自 2021 年后没有持续维护。恢复游戏策略开发前，下一阶段需要更强的工程约束、可测试性和文档。

## 决策

删除旧代码，从空的 TypeScript Screeps 代码库重建。保留 Git 历史，初始化 Trellis，并只添加验证构建和测试基础设施所需的最小运行时 loop。

## 后果

- 删除旧 role 文件、旧 Rollup 配置、旧 npm lockfile 和旧测试。
- 新开发从已记录的 TDD 工作流开始。
- 未来策略模块必须通过测试和任务本地验收标准证明其公开接口合理。
