# PRD: Mature Bot Bunker Layout Survey

## Goal

Research mature Screeps bunker/base-planning patterns and translate them into local design constraints for an aggressive road-lattice base rebuild.

## Scope

- Inspect mature sources for layout concepts: bunker anchors, road lattices, extension gardens, open diagonals, traffic control, migration sequencing, tower/core placement.
- Record license posture and whether each concept is safe to adopt as an idea only.
- Produce a local recommendation matrix: adopt now, defer, reject.

## Non-Goals

- No source-code copying from mature bots.
- No local implementation or live operations.

## Acceptance Criteria

- Research notes cite concrete sources/paths/links.
- Output distinguishes concept adoption from code reuse.
- Recommendations are mapped to P2-P6 child tasks.
